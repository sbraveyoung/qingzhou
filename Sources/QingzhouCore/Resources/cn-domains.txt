# 来源：v2fly/domain-list-community（geosite:cn = tld-cn + geolocation-cn）的高频子集，
#      人工按「国内主流互联网服务 + 其 CDN/静态资源域」挑选，
#      并用 scripts/update-cn-domains.py 逐条校验仍在上游 geosite:cn 内。
# 更新：python3 scripts/update-cn-domains.py  —— 拉取上游、递归解析 include，
#      报告本表中已不在 geosite:cn 的条目（删掉）；新增条目手工追加后重跑校验。
# 约定：一行一个 registrable domain（后缀匹配，子域名自动覆盖）；全小写；
#      **不收 .cn / .中国 结尾的域名** —— 代码里的 TLD 规则已整体覆盖，收了是冗余。
# 规模：几百条、几 KB，远小于 100KB 预算；宁缺毋滥，拿不准归属的不收。
